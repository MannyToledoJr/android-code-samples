About
====================

这是一个面向初学，介绍 Android 应用开发中基础原理的的一个 repository ，里面涵盖了大多数 Android 开发过程中需要用到的基本代码。代码整体上以讲述基本原理为主，一方面为了用代码的方式阐明 Android 相关子模块的工作原理，另一方面为在实际开发中遇到类似的需求时提供代码参考或者直接粘贴使用

Build
====================

下载本仓库后根目录中文件夹下都是一个完成的 Eclipse 工程，直接导入即可
